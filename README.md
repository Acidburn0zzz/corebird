
# Corebird

## Features
 - Support for multiple accounts
 - Direct Messages(in development)
 - Inline media with support for various sites
 - bla bla try it out.


## Dependencies
 - gtk+-3.0 >= 3.9
 - glib-2.0 >= 2.38
 - rest-0.7
 - json-glib-1.0
 - libnotify
 - sqlite3
 - libsoup-2.4
 - vala >= 0.22 (makedep)
 - cmake >= 2.6 (makedep)

## Compilation

```
cmake . -DCMAKE_INSTALL_PREFIX=/usr
make
make install
```
